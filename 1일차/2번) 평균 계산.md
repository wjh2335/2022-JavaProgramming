## main.java 소스코드
```java
package pack1;
import java.util.Scanner;
public class main
{
	public static void main(String[] args)
	{
		Scanner input = new Scanner(System.in);
		
		int n = input.nextInt();
		double sum = 0, max = 0;
		
		for(int i = 0; i < n; ++i)
		{
			int a = input.nextInt();
			sum += a;
			if(max < a) max = a;
		}
		
		System.out.println(sum / max * 100 / n);
		input.close();
	}
}
```
